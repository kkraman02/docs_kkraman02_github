Neuron Startup Menu¶

Neuron Startup Menu is an useful tool to help ROS environment setup.

When it comes to configuring ROS environment, there are several things to be considered. For example:

  • Installing ROS 1, ROS 2, DDS, and other useful packages.

  • Need to setup the environment everytime you open a new terminal.

  • The dependency between ROS and Ubuntu version.

To solve these problem, Neuron Startup Menu has the following features:

  • Installing ROS environment in one command.

  • Provide menu to choose ROS environment for each new terminal.

  • Able to run ROS in docker container. It’s useful if you can’t change the Ubuntu version of your platform.
